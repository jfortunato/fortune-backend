Get Started

This configuration has already been setup as a demo. Navigate to /dogs to see it in action.

Configuring Resources

Adding your resources to backend/config/resources.yml...

# resources.yml
---

dogs: ~

puppies:
    parent: dogs

...will produce the following RESTful routes

# dogs                          # puppies
---                             ---
GET    /dogs                    GET    /dogs/:dog_id/puppies
GET    /dogs/:id                GET    /dogs/:dog_id/puppies/:id
POST   /dogs                    POST   /dogs/:dog_id/puppies
PUT    /dogs/:id                PUT    /dogs/:dog_id/puppies/:id
DELETE /dogs/:id                DELETE /dogs/:dog_id/puppies/:id

Setup Database

You can use any database supported by PHP PDO, including MySQL. To keep things simple, this demo has setup an SQLite database located at backend/config/database/db.sqlite. You can use any sqlite administration tool to directly edit the database schema.

Build Your Application

If you are building a single page application using javascript, you can start to build your frontend. You will most likely start by replacing this file, which is located at public/templates/main.html with whatever your app requires.
